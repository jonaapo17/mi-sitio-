Glosario de Etiquetas HTML

Etiqueta Uso Ejemplo
<header> La cabecera, contiene la introduccion, el logo, titulo, o la navegacion principal <header>...</header>
<main> El contenido principal, solo debe haber uno por pagina <main>...</main>
<nav> Menu de navegacion, agrupa enlaces principales para moverte en un sitio <nav<a href="inicio"<inicio></a<
<footer> El pie de pagina, contiene informacion secundaria, avisos legales etc... <header>...</header>
<h1> a <h6> Permiten poner diferentes tamaños a los encabezados <h1>Amor eterno</h1>
<p> Sirve para poner parrafos en una pagina web <p>Te amo mucho</p>
<b> Es para indicar que el texto esta en negrita <p>Luis<b>Miguel</b></p>
<i> Es para indicar que el texto esta en cursiva <p>Amarte<i>Duele</i></p>
<sub> Es para indicar que el texto esta en subindice <p>Alfredo<sub>Adame</sub></p>
<sup> Es para indicar que el texto esta en superindice <p>Jose<sup>Jose</sup></p>
<q> Define una cita corta para que sea una sola linea <p><q>Esto es una sola linea</q></p>
<blockquote> Sirve para extraer informacion que no es nuestra <blockquote>"Una en el dia y la otra en la noche"</blockquote>
<abbr> Define abreviaturas o acronimos <p>El<abbr>title=Instituto Nacional Electoral"<INE</abbr>es bueno</p>
<br> Sirve para dar un salto de pagina Hola<br>Mundo;
<address> Para informacion de contactos <address>Escrito por Monse<br>Visitanos</address>
<cite> Poner una obra creativa como pelicula, pinturas etc... <p><cite>The Scream</cite>by Edward</p>
<title> Nos sirve para colocar el titulo <title>Amores Perros<title>
<img> Sirve para colocar una imagen en la pagina web <img scr="Ya chucky vete no.jpg">
<body> Contiene el contenido visible de la pagina <body>...</body>
<html> Contiene todo el documento de HTML <html>...<html>
<ul> Lista no ordenada(con viñetas) <ul><li>Elemento</li></ul>
<ol> Lista ordenada(numerada) <ol><li>Elemento</li></ol>
<li> Elemento de una lista <li>Elemento</li>
<!DOCTYPE html> Indica que el documento es HTML <!DOCTYPE hmtl>
<a> Sirve para colocar enlaces o hipervinculos <a href="">Google</a>
